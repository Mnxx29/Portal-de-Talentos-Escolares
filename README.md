# 🎓 Portal de Talentos Escolares

## 📌 Descripción

Este proyecto consiste en la creación de un sitio web moderno, desarrollado con HTML5, CSS3, Bootstrap y JavaScript/jQuery, cuyo objetivo es dar a conocer y destacar los talentos de estudiantes en las áreas de **música** y **deportes**.

A través de este portal, se busca motivar la participación y reconocer a alumnos destacados, permitiendo explorar distintas categorías, acceder a fichas individuales con información detallada de cada talento, y permitiendo la postulación de nuevos talentos a través de un formulario interactivo.

---

## ✨ Características Destacadas

* **Diseño Moderno (Glassmorphism):** Interfaz visual atractiva que utiliza efectos de desenfoque de fondo y transparencias para simular cristal en las tarjetas y la barra de navegación.
* **Diseño Responsivo:** Adaptable a dispositivos móviles, tablets y computadoras de escritorio gracias a la integración de Bootstrap 5.
* **Formulario Validado:** Un formulario de postulación robusto cuyas entradas (nombres, correos, números telefónicos) son validadas en tiempo real utilizando expresiones regulares (Regex) y jQuery.
* **Estructura Modular y Buenas Prácticas:** Uso de estilos CSS bien organizados, bibliotecas externas (como Bootstrap Icons y Google Fonts), y etiquetas `<head>` estandarizadas (como las `meta description` para SEO básico).
* **Footers Interactivos:** Pie de página cohesivo en todo el portal con enlaces simulados a redes sociales institucionales.

---

## 🧩 Estructura del Sitio

El sitio está compuesto por:

* **Página principal (`index.html`):**
  * Presentación general del portal (Hero Banner).
  * Carrusel interactivo de imágenes de estudiantes.
  * Lista y descripción de talentos destacados.

* **Páginas de categorías:**
  * `Música.html`
  * `Deportes.html`

* **Fichas individuales (Carpeta `Talentos/`):**
  * Perfil detallado de cada estudiante (Ej. Valentina Rojas, Matías Soto, etc.).
  * Muestra de sus logros, habilidades y galerías de imágenes.

* **Página de Postulación (`FormularioPostulacion.html`):**
  * Formulario completo con feedback de errores y validaciones manejadas por JavaScript.

## 🛠️ Tecnologías Utilizadas
