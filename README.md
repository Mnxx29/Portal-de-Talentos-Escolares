# 🎓 Portal de Talentos Escolares

## 📌 Descripción

Este proyecto consiste en la creación de un sitio web básico desarrollado en HTML5, cuyo objetivo es dar a conocer y destacar los talentos de estudiantes en las áreas de **música** y **deportes**.

A través de este portal, se busca motivar la participación y reconocer a alumnos destacados, permitiendo explorar distintas categorías y acceder a fichas individuales con información detallada de cada talento.

---

## 🧩 Estructura del Sitio

El sitio está compuesto por:

* **Página principal (index.html):**

  * Título del portal
  * Descripción general
  * Lista de talentos destacados
  * Imagen representativa
  * Enlaces a categorías

* **Páginas de categorías:**

  * Música
  * Deportes

* **Fichas individuales:**

  * Nombre del estudiante
  * Edad
  * Talento
  * Descripción
  * Imagen
  * Enlace de retorno
